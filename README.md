# Mantooth.E01 Digital Forensics Report

## Objective

The Mantooth.E01 project aimed to perform a forensic examination of the Mantooth computer hard drive image. The focus was on uncovering deleted files, suspicious activity, and evidence of criminal behavior. This project provided practical experience with forensic tools, evidence validation, and documentation in a criminal investigation scenario.

### Skills Learned

- Conducting forensic analysis of disk images related to criminal investigations.
- Using Autopsy and Exterro FTK Imager for digital evidence collection and verification.
- Recovering deleted files and analyzing email, web history, and system logs.
- Identifying suspicious software, web searches, and images linked to illegal activity.
- Creating a structured timeline of digital evidence and presenting findings clearly.
